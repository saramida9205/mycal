# My일정관리 (MyCal) 📅

**My일정관리**는 사용자 친화적인 디자인과 강력한 일정 연동 기능을 갖춘 PHP 기반의 현대적인 웹 캘린더 애플리케이션입니다. 개인의 일정을 효율적으로 관리하고, 외부 캘린더와의 호환성을 완벽하게 지원합니다.

---

## ✨ 주요 기능

### 1. 사용자 경험 및 디자인 (UX/UI)
- **프리미엄 UI/UX**: 모던한 디자인 시스템과 부드러운 애니메이션이 적용되었습니다.
- **다크 모드 (Dark Mode)**: 눈의 피로를 덜어주는 어두운 테마를 지원하며, 시스템 설정 및 수동 전환이 가능합니다.
- **반응형 레이아웃**: 모바일 기기에서도 최적화된 보기 모드(목록 보기 등)를 자동으로 지원합니다.
- **실시간 검색**: 일정 제목 및 본문 내용을 즉시 검색하여 필터링할 수 있습니다.
- **사이드바 유틸리티**: 미니 캘린더, D-Day 알림 섹션, 관리 도구가 집약된 편리한 사이드바를 제공합니다.

### 2. 일정 및 데이터 연동
- **ICS 가져오기/내보내기**: Google 캘린더, Outlook 등 외부 서비스의 일정(.ics)을 연동하거나 백업할 수 있습니다.
- **반복 일정 지원**: 매일, 매주, 매월, 매년 반복되는 스마트 일정을 지원합니다.
- **공휴일 자동 동기화**: 한국의 주요 공휴일이 달력에 자동으로 표시됩니다.
- **카테고리 시스템**: 업무, 개인, 중요 등 카테고리별 색상 지정 및 D-Day 자동 관리가 가능합니다.

### 3. 관리 및 보안
- **DB 백업 및 초기화**: 전체 데이터를 SQL 파일로 저장하거나, 한 번에 초기화할 수 있는 관리 도구를 제공합니다.
- **세션 보안 강화**: 세션 관리 최적화 및 보안 헤더(X-Frame-Options 등)를 적용하여 안전한 환경을 구축했습니다.
- **로그인 시스템**: 비밀번호 기반의 인증 시스템과 안전한 로그아웃 기능을 제공합니다.

---

## 🛠 기술 스택

- **Backend**: PHP 7.4+, MySQL (PDO)
- **Frontend**: FullCalendar 5.11, TinyMCE (Editor), Vanilla JS, Custom CSS Variables
- **Architecture**: RESTful API 기반의 데이터 연동

---

## 📂 프로젝트 구조

```text
/mycal
├── public/                 # 웹 접근 경로
│   ├── api/                # 백엔드 API 엔드포인트
│   ├── css/                # 스타일시트 (디자인 시스템)
│   ├── js/                 # 프런트엔드 로직
│   ├── login.php           # 로그인 페이지
│   └── index.php           # 메인 애플리케이션 화면
├── src/                    # 비즈니스 로직 및 라이브러리 (Auth 등)
├── config/                 # 데이터베이스 및 서버 설정
└── README.md               # 프로젝트 문서
```

---

## 🚀 빠른 시작

1. **데이터베이스 설정**: `config/database.php`에 DB 접속 정보를 입력합니다.
2. **테이블 생성**: `events` 테이블을 생성하고 필요한 컬럼(`category`, `recurrence_rule`, `color` 등)을 추가합니다.
3. **접속**: `public/login.php`를 통해 로그인 후 일정을 관리하세요.
